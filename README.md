# Beyond Backscatter: InSAR Coherence from Detected SAR Images

Deep learning framework for estimating InSAR coherence directly from detected SAR imagery, enabling large-scale coherence generation from SAR products available through Google Earth Engine.

This repository will contain the code, experiments, notebooks, pre-trained models, and presentation slides accompanying the paper *"Beyond Backscatter: InSAR Coherence from Detected SAR Images"*.

🚧 **Work in progress:** the repository is currently under construction, and the source code and trained models will be released soon.

📑 **Presentation slides:** available in the `slides/` directory of this repository.
